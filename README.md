# Django Entity Mapping System

This project implements a modular Django REST API with the following entities:

Master Entities
- Vendor
- Product
- Course
- Certification

Mapping Entities
- Vendor → Product
- Product → Course
- Course → Certification

Technologies Used
- Django
- Django REST Framework
- APIView
- drf-yasg (Swagger)
